# CodExpert AI

A polished, single-page assistant experience for shipping code faster. Paste snippets, capture bugs, or plan features and let CodExpert AI respond with focused, high-quality guidance. The refreshed UI includes a glassmorphic layout, responsive conversation history, and inline code highlighting with quick copy actions.

## Features
- Modern glass UI with floating gradients and responsive layout
- Multi-model selector with persistent conversations stored in local storage
- Streaming responses with inline code formatting and one-click copy buttons
- Keyboard-friendly input (Shift+Enter for new line) and quick `/name` personalization
